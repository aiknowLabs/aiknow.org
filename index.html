Job Summary
We are looking for a skilled React Native developer for developing native apps on Android as well as IOS. The candidate should have at least 2-year experience and should have the knowledge and confidence to work solo (independently) in projects and be able to deliver in the committed timeline.
Qualifications and Skills
Essential:
B.E. / B.Tech / MCA with 2-3 years of experience of proven UI/UX Development
Responsibilities and Duties
  • Ability to think from end user’s perspective and focused on improving the overall product experience
  • Very good understanding of the React Native framework. 1 year of experience in React Native based application development.
  • Hands-on experience on React Native APIs, ReactJS, Javascript
  • Hands-on experience on Web development technology stack, HTML/JS/CSS and other web development libraries and frameworks is an added advantage
  • Ability to coach teams in writing quality code with documentation
  • Thorough understanding of React Native development tools like IDEs, Chrome debugging tools, ESLint, React Native CLI, Watchman, package manager, etc
  • Understanding of Android and iOS Code signing process and deployment model
  • Excellent expose and expertise in design patterns and software design patterns in general.
  • Ability to write unit test cases and test their code accordingly.

  • React Native, JavaScript, Android, and IOS Development, HTML5
